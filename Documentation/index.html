What's included? - top

Flare ui is an elegant Activity and Fitness App ui comes with 13 screens, 11 reusable Widget



Requirements - top

  • Any Operating System (ie. MacOS X, Linux, Windows)
  • Any IDE with Flutter SDK installed (ie. IntelliJ, Android Studio, VSCode etc)
  • A little knowledge of Dart and Flutter


Instructions - top

  • Create a new folder ex: flare_ui.
  • Extract the zip inside it.
  • Use your favorite IDE and navigate to project (VS Code, Android studio..)
  • Run "flutter pub get" to download missing dependecies.
  • Replace your API key in the application manifest: android/app/src/main/AndroidManifest.xml
  • Replace your API key in the application delegate: ios/Runner/AppDelegate.m or if you are using swift code: ios/Runner/AppDelegate.swift
  • that's it!
